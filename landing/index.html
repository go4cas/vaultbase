get started

Three commands. No setup.

Drop the binary on any box and run it. The first launch opens a setup wizard — pick an admin, and every gear is already turning.

install · one line signature-verified
$ curl -fsSL https://get.cogworks.dev | sh
✓ resolved v1.0.0
✓ cosign verify OK · sigstore
✓ installed → /usr/local/bin/cogworks
$ cogworks
# → http://localhost:8090/_/setup setup wizard
build from source bun
$ git clone github.com/go4cas/cogworks
$ cd cogworks && bun install
$ bun run build → ./cogworks
$ ./cogworks
✓ one binary · zero deps · macos / linux / windows
# first run launches the setup wizard
